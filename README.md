# SpaceX Falcon 9 First Stage Landing Prediction

Performed Exploratory Data Analysis (EDA) and Feature Engineering on SpaceX Falcon 9 launch data.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Tasks
- Visualized launch success trends
- Analyzed payload mass vs launch outcomes
- Examined launch site and orbit relationships
- Created categorical dummy variables
- Prepared dataset for machine learning models

## Dataset
SpaceX launch records provided in IBM Applied Data Science Capstone.

## Course
IBM Applied Data Science Capstone (Coursera)
